iCatalog

Introduction to Parse framework

This project is designed for anyone who wants to get started with a back-end database handling, in this case creating a user and dealing with signup and login process. In this app you will learn how to handle these features using a very powerful framework called Parse. Further more you will experience a little with Mapkit framework. Check my website at www.siavashalipour.com for more information about the tutorials.

Note: Parse framework is a property of Parse (www.Parse.com)
